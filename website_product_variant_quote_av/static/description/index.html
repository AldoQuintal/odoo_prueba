Using this Odoo Apps you can get the quotation from the website. There is configuration in product to select product you want to not sell direct from the shop. Quote product price is hide from the shop page and product details page.

You can get notification email of request quote and also there is an configuration if you want to send automatical quotation when user request for quote.

Features

Using this Odoo Apps you can get quotation from the website based on the configured products.
You can select product that you want for quotation
There is configuration in product to select product you want to not sell direct from the shop.
Product price is hide from the shop and product details page
If product have variant then you can also add variant product to get quote
You can get notification email of request quote confirmation
There is an configuration if you want to send automatical quotation when user request for quote
Non login user also request for the quotation by share the some of the details
Seprate view in backend to find quotation
